## 1. 키워드
* __최적화__ - title 태그, meta 태그, 파일명, 폴더명, CSS 클래스명, time 태그, 꾸준한 업데이트
* __접근성__ - wai-aria, role, 탭 이벤트, 탭 포커스
* __마크업__ - 시맨틱, 정보에 맞는 태그 이용
* __반응형__ 디자인  
<br/>

## 2. 사용한 언어
* HTML
* CSS(반응형 디자인 구현)
* Javascript
<br/>

## 3. 기록 사항
1. HTML 태그의 특징
2. CSS 속성 및 디자인 구현
3. Javascript 메소드와 객체 정리
4. git bash 명령어와 에러 대처 방법
5. 에디터(아톰) 명령어
<br/>

## 4. 페이지 구성
* 헤더
  * 홈으로 이동
  * 야간 모드로 전환
  * 네비게이션(페이지 이동)
* 메인
  * 공부한 내용
* 푸터
  * 정보의 출처
  * 이모티콘 출처
