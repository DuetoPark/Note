🚪 NVM 🚪

🔨 개요

개발 프로젝트에 따라 Node의 버전을 여러 개 설치하고, 번갈아 가면서 사용해야 할 때가 있다.
NVM(Node Version Manager)를 이용해서 Node 버전을 설치해서, 원하는 Node 버전을 골라서 사용할 수 있다.
nvm 사용법에 대해 알아보자.

📌 nvm 설치

  • curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.1/install.sh | bash
  • wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash

위의 명령어 중 하나를 명령한다.

nvm install version
새로운 nodejs 설치
nvm install v15.11.0
특정 버전의 nodejs 설치

📌 nvm 특정 Node 버전 활성화 하기

nvm use 15.11.0

📌 nvm 버전 확인

nvm version

📌 nvm 설치 리스트 확인

nvm ls
