💇 NPM 💇

👶 개요

npm 명령어를 알아봅시다.

🙈 명령어

npm install [package]
라이브러리/패키지 설치
npm i: npm install의 줄인 명령어
옵션
--save--dev
package.json 파일에 의존성 내용을 추가
devDependencies 객체에 패키지 정보 추가
--save
dependencies 객체에 패키지 정보 추가
--force
해당 패키지가 존재하더라도 원격 저장소에 있는 패키지를 가져온다.
-g
전역 폴더에 설치
위치: c:/사용자/win10/AppData/Roaming/npm/node_modules
npm ls
npm package 설치 리스트
옵션
-g
global(전역) 설치 확인
수많은 dependencies까지 보인다.
--depth=0
설치한 모듈만 확인
