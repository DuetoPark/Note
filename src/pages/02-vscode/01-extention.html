👧 개요

최근들어 아톰이 버벅이길래 vscode를 사용해봤다.
아직은 좋은지 아닌지 모르겠다. 빨리 익숙해져야지.
일단 extension들을 정리해놓으려 한다.

🙌 Extension

extension은 플러그인과 비슷하다.
폴더의 아이콘을 바꿔주거나, 괄호에 색깔을 입혀서 매칭되는 괄호를 시각적으로 강조하거나 등의 역할을 가진다.
정말 많은데 그 중 많이 사용하는 extension 리스트를 적어놓으려 한다.

👉 Extension 검색

화면 왼쪽 상단 5번째 아이콘
화면 왼쪽 하단 톱니바퀴-Extensions
단축키: shift+ctrl-X

👉 Extension 목록

  • Meterial theme
  • Meterial icon theme
  • Auto Rename Tag
  • HTML to CSS auto completion
  • HTML CSS Support
  • CSS Peak
  • Autoprefixer
  • Bracket Pair Clorizer 2
  • Indent-rainbow
  • Live Server
  • Prettier

👉 Autoprefixer 실행

파일선택-톱니바퀴-command palette...-Autoprefixer:run
(ctrl+shift+P)-Autoprefixer:run

👉 Live Server 실행

파일선택-톱니바퀴-command palette...-Live Server: Open with live server
(ctrl+shift+P)-Live Server: Open with live server
