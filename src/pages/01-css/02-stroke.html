⚠️ 텍스트 외곽선 ⚠️

🎉 stroke

텍스트에 외곽선을 넣는 이유는 텍스트를 강조하기 위함이다.
css의 stroke 속성를 이용한다.

🔥 대신, 표준방법이 아니라서 웹킷 및 모질라 계열의 브라우저에서만 적용된다. 🔥 

                  
                    -webkit-text-fill-color: white;//투명도 가능
                    -webkit-text-stroke: 1px #000;

🌏 웹 크로스 브라우징

text-shadow를 이용한다.

                  
                    .text-stroke {
                      color: #f00;
                      text-shadow:
                      -1px -1px 0 #000,
                      1px -1px 0 #000,
                      -1px 1px 0 #000,
                      1px 1px 0 #000;
                    }

🔇 애니메이션

색만 바뀌고 두께는 변하지 않는다.

                  
                    /* Only the color will change, not the width */
                    @keyframes colorchange {
                      0% {-webkit-text-stroke: 10px red;}
                      100% {-webkit-text-stroke: 20px green;}
                    }
