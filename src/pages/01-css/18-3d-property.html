👋 3D Property 👋

🙌 개요

카드 넘기는 효과(flip)을 구현하면서 3D 표현하는 속성을 알게되었다.
속성에 대해 알아보자.

💁 프로퍼티 종류

perspective
원근법과 시점을 설정하는 속성
값이 적을수록 입체적이고 값이 커질수록 평면에 가까워진다.
transform-style
자식요소를 평면 또는 입체 공간에 둘 지 결정하는 속성
자식요소를 감싸는 부모요소에 선언한다.
속성값 flat - 자식요소를 평면에 둔다.
속성값 preserve-3d - 자식요소를 입체 공간에 둔다.
backface-visibility
요소의 뒷면이 사용자를 향할 때 보일지 말지 결정하는 속성
속성값 visible - 뒷면이 보인다.
속성값 hidden - 뒷면이 보이지 않는다.
육면체를 구현할 때 backface-visibility: hidden; 처리하면 육면체의 덩어리감이 부각된다.
