😇 main 태그 margin 😇

😈 개요

클론 코딩을 하던 중, 문득 궁금해졌다.
크롬은 잘 나오는데 인터넷 익스플로어는 잘 나올까?
내 페이지는 의외로 멀쩡했다.
main 태그에 max-width 안 먹힌거 빼고

인터넷 익스플로어의 메인태그에 max-width가 먹히지 않는 이유를 알아보자!

😎 IE와 main 태그

결론부터 말하자면, max-width문제가 아니었다.

IE11가 main 태그를 지원하지 않았기 때문이었다.
해결방법의 다음과 같다.

  • main 태그를 div 태그로 변경하거나
  • main{display: block;}을 선언한다.
