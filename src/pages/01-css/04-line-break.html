💜 줄 바꿈, 여백 처리 💜

🙏 줄바꿈 속성 종류

  • 줄바꿈 되지 않고 터져 나가는 url
  • 댕강 썰리는 한국어 단어
  • 그대로 노출되는 연속 스페이스

궁금했다. 나. 이것도 처리 가능한지. css의 속성으로. 미안한 해석.
구글번역을 사용했다.
마침내 나는 찾았다 그것을. 종류가 여러개.

🙏 줄바꿈 속성 종류

  1. word-wrap
  2. word-break
  3. white-space

🙏 word-wrap

word-wrap
띄어쓰기가 없는 긴 텍스트를 어떻게 처리할 지 지정
대상: 비아시아 언어(영어, 아랍어 등등)
주소같은 문자열을 끊음
(참고) <cite> 태그에 입력한 url은 기본적으로 줄바꿈이 되지 않는다.
속성
  • normal
  • break-word
  • initial
  • inherit
word-wrap이 가능한 요소
인라인 요소
블록요소
절대적 위치(position: absolute;)의 블록요소

🙏 word-wrap 값

normal
단어가 너무 길 때
요소의 너비를 뚫고 나갈 때
단어를 뽀갬
break-word
문장이 너무 길 때
단어가 너무 길 때
break point(엔터, 띄어쓰기)에서 줄바꿈
단어를 뽀갬
initial
기본값으로 설정
inherit
부모 요소의 속성값을 상속

overflow-wrap 속성은 부모요소로부터 자식요소에게 상속된다.

💫 word-break

word-break
띄어쓰기가 없는 긴 텍스트를 어떻게 처리할 지 지정
대상: 아시아, 비아시아 언어

💫 word-break 값

normal
아시아 언어의 기본 설정값
단어가 너무 길 때
단어를 뽀갬
break point(엔터, 띄어쓰기)에서 줄바꿈 없음
break point(탭, 엔터) 축약
keep-all
문장이 너무 길 때
break point(엔터, 띄어쓰기)에서 줄바꿈
break-all
단어가 너무 길 때
단어를 뽀갬
break point(엔터, 띄어쓰기)에서 줄바꿈 없음
break point(탭, 엔터)는 축약

🍍 white-space

nomal
연속 스페이스 병합
엔터는 스페이스처럼 처리
문장이 길어서 튀어나가면 줄 바꿈
nowrap
연속 스페이스 병합
줄바꿈: <br>
문장이 길어서 튀어나가도 줄 바꿈 안 함
pre
연속 스페이스 유지
줄바꿈: 엔터, <br>
문장이 길어서 튀어나가도 줄 바꿈 안 함
pre-wrap
연속 스페이스 유지
줄바꿈: 엔터, <br>
문장이 길어서 튀어나가면 줄 바꿈
pre-line
연속 스페이스를 병합
줄바꿈: 엔터, <br>
자동 줄바꿈
break-spaces
연속 스페이스 유지(문장 끝에 위치한 연속 스페이스 포함)
줄바꿈: 엔터, <br>, 연속 스페이스의 중간 또는 끝
문장이 길어서 튀어나가면 줄 바꿈

🍋 white-space 요약

속성값 엔터(개행문자) 스페이스, 탭 자동 줄 바꿈 줄 끝의 공백
normal 병합 병합 예 제거
nowrap 병합 병합 아니오 제거
pre 유지 유지 아니오 유지
pre-wrap 유지 유지 예 넘침
pre-line 유지 병합 예 제거
break-spaces 유지 유지 예 줄 바꿈

hyphens

텍스트 해체쇼로 인해, 여러 줄에 걸치는 단어에 붙임표를 추가하는 방식

none
단어 뽀개지 않음
줄바꿈은 공백 문자에서만 발생
manual
단어 내의 줄 바꿈 위치를 나타내는 문자에서 줄바꿈
auto
overflow-wrap: break-word;와 같음
