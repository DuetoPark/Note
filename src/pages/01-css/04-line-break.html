🙏 줄바꿈 속성 종류

내 페이지를 어지럽히는 몇몇 친구들. 절교하자.

  • 줄바꿈 되지 않고 터져 나가는 url
  • 댕강 썰리는 한국어 단어
  • 그대로 노출되는 연속 스페이스

궁금했다. 나. 이것도 처리 가능한지. css의 속성으로. 미안한 해석. 구글번역 사용했다.
마침내 나는 찾았다 그것을. 종류가 여러개.

  1. word-wrap
  2. word-break
  3. white-space

🙏word-wrap

word-wrap 속성은 띄어쓰기가 없는 긴 텍스트를 어떻게 처리할 지 지정하는 속성이다.

word-wrap는 주소같은 문자열을 끊을 때 사용한다.
(참고, <cite> 태그에 입력한 url은 별다른 스타일을 주지 않으면, 기본적으로 줄바꿈이 되지 않는다.)

주로 비아시아 언어의 줄바꿈을 제어한다.
아시아언어(한국어, 중국어, 일본어 등등)의 줄바꿈은 word-break속성을 이용한다.

👋word-wrap이 가능한 요소
  • 인라인 요소
  • 블록요소
  • 절대적 위치(position: absolute;)의 블록요소

break-spaces
pre-wrap과 거의 동일
👙break-spaces와 pre-wrap의 차이
  • 연속 공백이 줄의 끝에 위치하더라도 공간을 차지한다.
  • 연속 공백의 중간과 끝에서도 자동으로 줄을 바꿀 수 있다.
  • 유지한 연속 공백은 pre-wrap과 달리 요소 바깥으로 넘치지 않으며, 공간도 차지하므로 박스의 본질 크기(min-content, max-content)에 영향을 준다.
