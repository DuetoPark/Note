🌟 개요

Bootstrap에서 container의 width를 조정하다가 삽질한 내용을 정리해보려 한다.

💚 Width

기본적으로 Bootstrap container는 viewport마다 정해진 width가 있다. 이 width는 임의로 조정 가능하다.

👉 원인

Bootstrap의 container 자체 스타일
(min-width 프로퍼티에 값과 !important 선언되어 있음)

👉 해결방법

min-width 프로퍼티에 원하는 값과 !important를 함께 선언

strong CSS의 규칙에 따라, !important도 최신에 선언된 CSS 스타일을 적용한다.
