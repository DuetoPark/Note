🌟 개요

클론 코딩중인데,
breakpoint 확인하려고 화면 너비 늘렸다 줄였다 반복하다가 발견한게 있다.

뷰포트의 가로 길이가 변하면, 요소의 여백은 유지되고 요소의 너비만 조절되는 매직 쇼.

마법의 주문.. 아직 배워야 할 게 넘모 많습니다..

✨ width: calc();

어떻게 한거지? 이렇게 저렇게 계속 찾아보고 시도해보다가 calc를 사용했다는 걸 알게되었다.

calc에는 연산자를 다 쓸 수 있다.
대신, 연산자는 좌우 공백이 있어야 한다.

width: calc((100% - 64px) / 3); 이렇게 사용한다.
여백 계산방법은 다음과 같다.

  1. 먼저, 전체 가로 길이(100% 또는 100vw)에서 요소의 여백의 합계를 뺀다.
  2. 다음, 요소 갯수만큼 나눈다.

🎵 width: auto/100%;

그리고 width에 auto나 100%같이 유동적인 값을 줄 땐, 확인할 부분이 많다.

뭔가 width 조정이 잘 안 될 때가 있는데,
일단 화내지 말고 여러가지 확인을 해야한다.

이 그룹의 width를 결정하는 요소는 어디에 있나?를 찾기 위함이다.

  • 자식요소의 display와 width
  • 해당요소의 display와 width
  • 부모요소의 display와 width

어느 요소에 영향을 받는지 확인했다면, 결론은 다음과 같다.

  1. 해당요소가 {width: 100%;}이면, 자식요소의 width가 그룹의 크기를 조절한다.
  2. 자식요소가 {width: 100%;}이면, 해당요소의 width가 그룹의 크기를 조절한다.
