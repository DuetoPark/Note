☁️ Gitignore ☁️

⚡ 개요

원격 저장소에서 관리하지 말았으면 하는 파일들이 있다.
가끔 페이지 구상을 파워포인트로 할 때가 있다.
구상 파일을 작업 폴더에 넣어놓고 싶은데, 원격 저장소에 올라가지 않았으면 한다.
이런 파일들을 git에서 추적하지 않게 만드는 장치가 gitignore이다.

🌊 gitignore 구현

(참고) 전문적으로 'git에서 파일 트랙킹을 멈춘다'라고 표현한다.

(참고) 파일을 .gitignore에 설정하면, 파일의 업로드/수정/제거/커밋 등을 무시한다.

  1. 작업 폴더에 .gitignore 파일 생성
  2. .gitignore 파일에 원하는 파일/폴더명 입력
  3. git rm --cached -- 파일/폴더 경로
  4. git add .
  5. git commit 커밋 메시지
  6. git push -u origin {브랜치명}

⛄ 작성 패턴

*.class

!

-g
전역 폴더에 설치
