👽 Git Config 👽

작성 날짜: 2021년 06월 15일

🐴 개요

github 계정을 정리했다.
setting에서 계정 이메일을 변경하고, 로컬의 이메일도 수정했다.
로컬 이메일은 어떻게 수정하는지 알아보자.

🎉 git config

(참고) 이 방법은 깃허브 계정에 등록된 email 주소만을 바꾸는 과정입니다. 계정 자체를 변경하고자 한다면 이 글을 참고해주세요. git 계정 변경

(참고) github 이메일 계정 관리: [Github 페이지]-[Settings]-[Emails]

(참고) 로컬 이메일 수정: 프로젝트 폴더 git bash 또는 terminal

git config user.email

프로젝트에 등록된 email 확인

git config --global user.email 이메일주소

Github 계정에 등록된 주소 중에 원하는 메일 주소 입력
제발!! Github 계정에 등록한 이메일만 사용

👻 결과

깃 이메일(user.email)을 수정해도 잔디밭은 그대로 유지된다. 계정(user.name)을 변경하지 않았기 때문이다.

push 했는데 레포지토리에 커밋이 등록되지 않는다면, 수정한 이메일 주소를 확인하자. 주소에 오타가 있으면, 레포지토리에 제대로 반영되지 않는다.
