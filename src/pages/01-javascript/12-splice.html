🍠 slice 🍠

🍏 개요

배열을 쓱싹 자르면 원하는 배열을 만들 수 있다.
하지만 중간 부분을 빼내고 다른 요소를 추가해주기에는 많은 동작이 필요하다.
이걸 단 한줄로 줄여주는 메소드가 있다.
splice 메소드에 대해 알아보자.

🍇 정의

splice
배열의 기존 요소를 삭제 또는 교체하거나 새 요소를 추가하여 배열의 내용을 변경하는 메서드
뽑혀나온 값들은 배열 형태
원본 배열은 변경된다.(요소를 뽑아내면 원본에서 삭제되고 새로운 배열로 생성됨)
배열.splice(시작 인덱스, 요소의 개수, 요소1, 요소2, ... , 요소n);
splice 시작 인덱스
배열에 변화를 줄 시작점
필수 기재 사항
splice 요소의 개수
삭제할 요소의 개수
선택적 기재 사항
생략하면 자동으로 0값이 설정되므로 아무런 요소도 제거되지 않는다.
splice 요소
시작점으로부터 추가할 요소를 원하는 만큼 입력
선택적 기재 사항

🍐 배열의 기존 요소 삭제

배열.splice(인덱스, 개수);
배열의 시작 인덱스부터 원하는 개수만큼 요소가 제거
개수에 음수 적으면, 아무 변화 없음
배열.splice(n);
n번 인덱스를 포함해서 이후의 모든 요소 제거
n이 음수이면 뒤에서부터 자름
배열.splice(Math.floor(Math.random * 9), 1)[1];
무작위로 하나의 요소를 뽑아서 제거
                    
                      //방법 1
                      let A = ["안", "녕", "오", "이", "친", "구"];
                      let B = A.splice(2, 2);
  
                      결과)
                      A = ["안", "녕", "친", "구"];
                      B = ["오", "이"];
  
                      //방법2
                      A = ["안", "녕", "오", "이", "친", "구"];
                      let C = A.splice(2);
                      let D = A.splice(-2);
    
                      결과) C = ["안", "녕"]; / D = ["친", "구"];

🍈 배열의 기존 요소에 추가

배열.splice(인덱스, 0, 요소1, 요소2, ... , 요소n);
시작 인덱스부터 요소 추가
                    
                      let A = ["안", "녕", "오", "이", "친", "구"];
                      A.splice(2, 0, "토마토");
        
                      결과) A = ["안", "녕", "토마토", "오", "이", "친", "구"];

🍍 배열의 기존 요소에 변경

배열.splice(인덱스, 제거할 요소 개수, 요소1, 요소2, ... , 요소n);
시작 인덱스부터 입력한 개수만큼 요소가 사라지고, 그 자리에 새로운 요소가 추가된다. 
                    
                      let A = ["안", "녕", "오", "이", "친", "구"];
                      A.splice(2, 2, "고", "구", "마");
        
                      결과) A = ["안", "녕", "고", "구", "마", "친", "구"];
