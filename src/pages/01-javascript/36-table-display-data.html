😂 표의 화면과 데이터 매칭 😂

💩 개요

표에서 데이터와 화면을 매칭시키려고만 하면 계속 삽질을 한다. 그래서 아예 정리해놓고 외우려고 한다. 또 이 거지같은 섬에 남겨지지 말아야지.

👽 매칭 방법

빈 배열 생성
표의 칸 데이터를 담아낼 빈 배열을 만든다.
tdData.push([]);
이 배열은 줄 단위로 이차원 배열이 된다.
첫 번째 반복문
표의 줄을 생성할 때마다 tdData 배열에 빈 배열을 만든다.
해당 배열은 줄 단위로 데이터를 구분하는 역할을 한다.
데이터 형태 
                        
                          console.log(tdData); // [ [], [], [] ]
      
                          // 원하는 결과
                          [
                            [1-1번칸, 1-2번칸, 1-3번칸],
                            [2-1번칸, 2-2번칸, 2-3번칸],
                            [3-1번칸, 3-2번칸, 3-3번칸]
                          ]
두 번째 반복문
tdData 배열에 값을 push 한다.
tdData 배열의 인덱스에 주의한다. 이 부분에서 계속 삽질했다.ㅠㅠ
데이터를 넣으려고 빈 배열을 만든거니까 인자는 j가 아니라 i여야 한다. 
                  
                    var tdData = []; // 데이터 저장할 배열
      
                    const table = document.querySelector('table');
      
                    for (let i = 0; i < 3; i+= 1) {
                      var tr = document.createElement('tr');
                      tdData.push([]); // 새로운 배열 생성
      
                      for (let j = 0; j < 3; j += 1) {
                        var td = document.createElement('td');
                        tr.appendChild(td);
                        tdData[i].push(td); // 인자가 i이다.
                      }
                    }
