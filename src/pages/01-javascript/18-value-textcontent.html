🐌 value와 textContent 🐌

수정 날짜: 2021년 03월 17일

🍍 개요

value와 textContent차이에 대해 알아보자.

🌝 value

간단하기 말하자면, value는 태그의 꺽쇠 안에 들어가는 속성의 텍스트를 의미한다.

input이나 버튼 요소에서 주로 사용하며, submit되면 value 속성의 값이 서버로 전달된다.

🐚 textContent

간단히 말하자면, 브라우저 화면에 표시되는 글자이다.
textContent는 태그의 꺽쇠 바깥에 위치한다.
