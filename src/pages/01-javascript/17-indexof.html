🎂 indexOf 🎂

🍞 개요

findIndex와 비슷한 indexOf를 알아보자.

🍘 정의

indexOf
배열에서 값과 같은 동일한 요소의 인덱스를 출력하는 메소드
찾는 값이 존재하면, 해당 요소의 인덱스가 출력
찾는 값이 없으면, -1이 출력
배열.indexOf(값);

🍧 예시코드 

                
                  const beasts = ['ant', 'bison', 'camel', 'duck', 'bison'];
    
                  console.log(beasts.indexOf('bison'));
                  // expected output: 1
    
                  console.log(beasts.indexOf('giraffe'));
                  // expected output: -1
