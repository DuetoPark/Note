🍐 append와 appendChild 🍐

🍒 개요

append와 appendChild의 차이. 언젠가는 정리해야지 했던 게 벌써 한 달이 지났다. 비슷한 맥락으로 묵혀놓았던 조각 개념들을 정리 중이다. 생각보다 시간이 걸린다. 돈은 티끌모아 티끌이면서. 다른건 꼭 이러더라. 구몬숙제부터 시작된 역사는 잘 바뀌지 않나보다.

🍆 append

append
노드 객체(Node object), 문자열(DOM String)을 인수로 받음
한 번에 여러 개의 자식 요소 설정 가능
return 값을 반환하지 않음
                    
                      div.append(p);
                      p.appned("고소한 맛이 나구")
  
                      // 브라우저 개발자 도구
                      // div
                      //  ㄴp("고소한 맛이 나구")
  
                      엉덩이.append(좌궁둥, 우궁둥);
      
                      // 브라우저 개발자 도구
                      // div#엉덩이
                      //  ㄴdiv.좌궁둥
                      //  ㄴdiv.우궁둥
  
                      // return
                        const 엉덩이 = document.querySelector("#wrapper");
      
                        console.log(엉덩이.append("방구장인")); //undefined

🍅 appendChild

appendChild
노드 객체(Node object)만을 인수로 받음
한 번에 하나의 자식 요소만 설정 가능
return 값을 반환
                    
                      // 노드 객체(Node object) 삽입
                        귤껍질.appendChild(알맹이);
      
                      // 브라우저 개발자 도구
                      // div#귤껍질
                      //  ㄴdiv.알맹이
  
                      엉덩이.appendChild(좌궁둥);
                      엉덩이.appendChild(우궁둥);
  
                      // return
                        const 엉덩이 = document.querySelector("#wrapper");
                        const 좌궁둥 = document.createElement("div");
                        좌궁둥.className = "뿡";
      
                        console.log(엉덩이.appendChild("좌궁둥")); // div.뿡
