🎊 this의 대상 🎊

🔕 개요

가끔 this를 써야하는데 엉뚱한 결과가 나온다.
콘솔로그하면 window가 나오거나 이벤트 적용대상이 나오곤 한다.
헷갈리니까 정리해봤다.

🎍 this가 가리키는 것

함수

객체(Object)

bind, call, apply
생성자(Constructor)

심화

'클릭 이벤트 속에 생성된 함수'의 this는 뭘까? 정답은 window다.
위에 정리했듯이, 함수의 this는 window다. 이벤트 리스너가 따로 처리를 못 한 모양이다.

이 상황에서 window말고 자기 자신을 가져오고 싶다면, 화살표 함수를 쓰면 된다. 

                  
                    const div = document.querySelector('div');
      
                    div.eventListener('click', function() {
                      console.log(this); // div
      
                      const test = function() {
                        console.log(this); // window
                      };
      
                      const inner = () => {
                        console.log(this); // div
                      };
                    });
