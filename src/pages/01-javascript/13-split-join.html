🚩 개요

숫자야구 게임 만들다가 나는 생각했다.
어.. 입력값하고 숫자배열 어떻게 구분하지
입력값은 숫자형이고 숫자배열은 배열이다.

String(숫자배열[0])+String(숫자배열[1])+...

찝찝하고 눈물나고 방구뀌고
이때 사용하는 배열/문자 변환 메소드가 split과 join이다.

💈 split 💈

🚧 정의

split
String객체를 지정한 구분자를 이용하여 여러 개의 문자열로 나누고 배열로 만드는 메서드
문자.split(구분자, 정수);
split 구분자
원본 문자열을 끝어야 할 부분을 의미
기호와 단어, 숫자, 두글자 이상의 문자열로 설정 가능
두글자 이상이면, 문자열 전체가 일치해야 함
생락하면, 원본 문자열 전체가 하나의 요소인 배열로 출력
split 정수
끊어진 문자열의 최대 개수를 나타내는 정수다.
배열의 원소가 정수에 도달하면 split 메서드의 동작을 멈춘다. 
                    
                      //구분자 생략
                      let 예제 = "오이친구".split();
    
                      결과: 예제 = ["오이친구"];
    
    
                      //구분자 입력
                      let 예제 = "오이친구".split("");
    
                      결과: 예제 = ["오", "이", "친", "구"];

                      //구분자가 중복
                      let 문자 = "https://www.example.com/get/thisfolder/thanks/html";
    
                      문자.split("/");
        
                      결과: ["https", "www.example.com", "get", "thisfolder", "thanks", "html"];

🎪 join 🎪

🚏 정의

join
배열을 문자열로 만드는 메서드
배열.join(구분자); 
                    
                      배열 = ["5", "2", "7", "9"];
        
                      배열.join(""); 결과) "5279"
                      배열.join(","); 결과) "5,2,7,9"
                      배열.join(":"); 결과) "5:2:7:9"
