📛 find 📛

📰 개요

Object.entires로 객체를 이차원 배열로 만들었으면, Object.keys[배열]을 사용하지 않고도 키를 가져다 쓸 수 있다.
해당 값이 들어간 배열을 찾아준다면, 배열을 찾고나면 키는 배열의 0 인덱스에 있으니까..!
이 때 사용하는 메소드가 find다.

📚 정의

find
배열에서 특정한 값이 해당된 객체나 배열을 찾아주는 메소드
find 자체가 반복문이다.
특정 값을 찾으면 return이 true가 되면서 동작을 멈춘다.
배열.find(콜백함수, thisArg);
find의 콜백함수
배열의 각 값에 실행할 함수
return 매개변수.키 === "찾고싶은 값";
find의 thisArg
생략가능하며, 콜백이 호출될 때 this로 사용할 객체이다. 
                    
                      var inventory = [
                          {name: 'apples', quantity: 2},
                          {name: 'bananas', quantity: 0},
                          {name: 'cherries', quantity: 5}
                      ];
        
                      function findCherries(fruit) {
                          return fruit.name === 'cherries';
                      }
        
                      console.log(inventory.find(findCherries));
                      // { name: 'cherries', quantity: 5 }

📇 findIndex

findIndex
찾고자 하는 요소의 위치를 출력하는 메소드
배열.indexOf(콜백함수);

📋 indexOf와의 차이

indexOf는 일차원 배열에 사용한다.
find와 findIndex는 이차원 배열에 사용한다.
