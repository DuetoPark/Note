💅 slice 💅

🍦 정의

slice
배열을 지정해놓은 시작점부터 끝점까지 잘라내여 복사본을 만드는 메소드
새로운 배열을 생성
원본 배열 유지
let 변수 = 배열.slice(시작 인덱스, 끝 인덱스);
slice 시작 인덱스
생략하면, 자동으로 0을 시작점으로 한다.
음수를 넣으면, 배열의 끝에서부터 시작한다.(slice(-2)는 배열에서 마지막 두개의 엘리먼트를 추출한다.)
배열의 길이보다 큰 숫자를 넣으면 빈 배열을 반환한다.
slice 끝 인덱스
추출을 종료할 기준 인덱스
end 인덱스를 제외하고 추출한다. 예를들어, slice(1, 4)는 두번째 요소와 세번째 요소만을 추출한다.
생략하면, 자동으로 배열 끝까지 추출한다.
음수를 넣으면, 배열 끝에서부터의 시작한다. slice(2, -1)은 세번째 요소부터 끝에서 두번째 요소까지 추출한다.

👓 sort 👓

🌂 정의

sort
배열의 요소를 아름답게 정렬하는 메소드
적용: 글자, 숫자
콜백함수를 지정하지 않으면, 숫자의 맨 앞숫자가 작은 순서대로 정렬( 12, 24, 28, 3, 48 이렇게)
배열.sort(콜백함수);
sort 콜백함수
return값이 0보다 크면 순서를 바꾼다.
음수를 넣으면, 배열의 끝에서부터 시작한다.(slice(-2)는 배열에서 마지막 두개의 엘리먼트를 추출한다.)
function compareNumber(a, b) {return a - b;} (오름차순)
function compareNumber(a, b) {return b - a;} (내림차순)
