❗ 정의

내장된 표현식을 허용하는 문자열을 의미한다.

❓ 문법 

            
              `string text`

              `string text line 1
               string text line 2`

              `string text ${expression} string text`

              tag `string text ${expression} string text`
  • 백틱(``)을 이용한다. (키보드 1번 왼쪽 키)
  • 플레이스 홀더를 이용하여 표현식을 넣을 수 있다. 달러와 중괄호로 표기한다. (${텍스트나 표현식})
  • 플레이스 홀더 안에서의 표현식과 그 사이의 텍스트는 함께 함수로 전달된다.
  • 코드가 읽기 쉽게 변한다.

💫 표현식 삽입법

일반 표현은 다음과 같다. 

            
              var a = 49;
              var b = 3;
              var 이름 = "오이";
              console.log("안녕 나는 숫자로 " + (a + b ) + " 야.\n한국어로는 " + 이름 + "라는 뜻이야. 까불지마.");
              // "안녕 나는 숫자로 52야.
              // 한국어로는 오이라는 뜻이야. 까불지마."

탬플릿 리터럴을 이용하면 읽기 쉬워진다. 위의 코드와 결과는 동일하지만 훨씬 직관적으로 변했다. 

            
              var a = 49;
              var b = 3;
              var 이름 = "오이";
              console.log(`안녕 나는 숫자로 ${a + b} 야.
              한국어로는 ${이름}라는 뜻이야. 까불지마.`);
              // "안녕 나는 숫자로 52야.
              // 한국어로는 오이라는 뜻이야. 까불지마."

💫 에러

  • 줄바꿈

    innerHTML 작성할 때, 백틱(`)입력 후에 줄바꿈하면 결과가 출력되지 않는다.
    백틱 뒤에 바로 입력해야 한다.

    일반 문자열은 줄바꿈해도 출력된다. 줄바꿈이 반영된다.
