📟 화살표 함수 📟

📡 개요

되게 많이 보이는데 모르겠는 표현.
대단하진 않지만 엄청난 화살표 함수에 대해 알아봅시다.

🔒 정의

화살표 함수
함수의 간결한 표현
구문이 짧고 항상 익명이다.
화살표 함수 인자
화살표 왼쪽에 선언
괄호 안에 선언
하나밖에 없을 때 괄호 생략
인자가 없으면 빈 괄호로 남겨둔다. 그녀석의 팔을 꼭 잡고 놓아주지 말자.
화살표 함수의 표현식
화살표 오른쪽에 선언
인자를 평가하고 평가 결과를 반환
중괄호 안에 선언
표현식이 한 줄이면 중괄호 생략
                    
                      /* 원본
                      const sum = function(a, b) {
                        return a + b;
                      };
                      */

                      // 화살표 함수
                      const sum = (a, b) => a + b;
                      alert( sum(1, 2) ); // 3

😊 예시

  • 매개변수가 하나

                        
                      const times = n => n * 2;
    
                      times(2); // 4

  • 매개변수가 둘 이상

                        
                      const sum = (a, b) => a + b;
    
                      sum(52, 79); // 131

  • 삼항연산자

                        
                      const cucumber = 52;
    
                      const check = (cucumber > 79) ?
                      () => alert('오이 친구 맞아요') :
                      () => alert('오이 친구 아닙니다');

  • 본문이 여러 줄인 화살표

                        
                      const sum = (a, b) => {
                        const result = a + b;
                        return result;
                      };
    
                      alert(sum(Math.ceil(Math.random * 10), 3)); // 0~10 아무숫자 + 3

😊 주의

메소드 함수가 아닌 곳에 가장 적합하며 생성자로서 사용할 수 없다.
