😐 스코프 😐

😇 개요

자바스크립트에서 무조건 알아야 하는 스코프, 실행콘텍스트, 프로토타입, 비동기, 클로저중에 스코프를 공부해보자.

💘 정의

스코프(scope)
변수 및 함수의 유효범위
범위: 에디터 창에서 함수가 접히는 만큼
함수 내부에 선언한 변수는 함수 밖으로 빠져나갈 수 없다. 
                  
                    var name = '나는 고구마';
                    function log() {
                      ver cucumber = '나는 오이';
                    }
      
                    console.log(cucumber);
                    // 참조에러 발생
                    // 함수 내부에서 선언된 변수이므로 함수 밖으로 나갈 수 없다.

💦 스코프 체인

스코프 체인
스코프 간의 상하관계
상위 함수의 변수: 하위 함수에서 사용 가능(컴퓨터는 실행하는 함수 안에 변수가 없으면 상위 함수로 거슬러 올라겨며 변수를 찾는다.)
하위 함수의 변수: 상위 함수에서 사용 불가능(변수는 함수 바깥으로 나올 수 없다.) 
                  
                    var name = '나는 고구마';
                    var tomato = '나는 토마토';
      
                    function log() {
                      name = '나는 오이';
                      var tomato = '나는 함수 내부에 선언한 변수라서 밖으로 못 나간다.';
                    }
      
                    // 전역 변수는 하위 함수에서 사용할 수 있다.
                    console.log(name); // 나는 오이
                    // 함수 안에 선언한 변수는 함수 밖으로 나올 수 없다.
                    console.log(tomato); // 나는 토마토

💩 렉시컬 스코프

렉시컬 스코프
함수 실행 위치가 아니라 선언된 위치에서 변수 찾는 것
결정: 코드가 적힌 순간
함수가 선언된 위치에서 변수를 찾는다.
함수가 실행된 자리에 함수를 대입하지 않는다.

✋ 렉시컬 스코프 예시 

                    
                      var name = '나는 오이';
      
                      function log() {
                      console.log(name);
                      }
      
                      function wrapper() {
                      name = '나는 고구마';
                      log();
                      }
      
                      wrapper(); // name = '나는 고구마';

함수가 선언된 위치에서 name 변수는 다음과 같다.

  • log 함수: 전역변수 name
  • wrapper 함수: 전역변수 name
                    
                      var name = '나는 오이';
      
                      function log() {
                        console.log(name);
                      }
      
                      function wrapper() {
                        var name = '나는 고구마';
                        log();
                      }
                      wrapper(); // name = '나는 오이';

함수가 선언된 위치에서 name 변수는 다음과 같다.

  • log 함수: 전역변수 name
  • wrapper 함수: 지역변수 name

렉시컬 스코프로 인해, wrapper안의 log가 실행되면, 전역변수 name에 저장된 '나는 오이'를 출력한다.
