👉 e.target 👈

🙌 개요

1분코딩 자바스크립트 강의를 듣다가 e.target이 활용도가 높아서 매우매우매우 편하다는 말씀을 듣고 얼마나 편한지 궁금해졌다.

그래서 javascript-3-copy.html 페이지에 클릭 이벤트를 추가해봤다.

🙏 이것만은 가져가고 만다

  • 개발자도구-property 활용법
  • 모던 javaScript 튜토리얼에서 배운 children과 parentElement 개념 정리

👸 이벤트 변수

이벤트 변수
마우스의 위치, 선택한 요소, 자식요소 등등의 정보가 담긴 변수
addEventListener에 지정한 handler함수에는 event 변수가 자동으로 생성된다.
event.currentTarget: 이벤트를 지정한 요소
event.target: 이벤트를 지정한 요소에서 클릭된 요소

🐲 개발자도구 property

이벤트 지정할 때 사용하기 좋은 정보들이 콸콸콸!

주로 확인하는 요소는 다음과 같다.

  • children
  • classList
  • lastElementChild
  • nextElementSibling
  • parentElement
  • previousElementSibling
previousElementSibling
선택한 요소의 바로 앞에 있는 요소
nextElementSibling
선택한 요소의 바로 뒤에 있는 요소
children
선택한 요소의 자식 요소
parentElement
선택한 요소의 부모 요소
