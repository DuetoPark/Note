🙌 Tab 컨트롤 🙌

🐰 개요

버튼을 누르면 특정 내용을 출력하는 동작 구현을 연습했다.
주로 메뉴-서브메뉴, 메뉴-본문관계를 생각하면 된다.
열심히 해봤다.

👽 [role=tab] 요소에 필요한 속성

role 속성
role='tab'
이 속성은 버튼-본문 관계에서 '버튼'에 해당하는 요소에 선언한다.
'뭐를 조작할 것이다'라는 의미이다.
aria-controls 속성
aria-controls='class/id'
버튼-본문 관계에서 '본문' 영역을 알려주는 속성이다. '뭐를 조작할 것이다'에서 '뭐'가 뭔 지 알려주는 장치이다.
본문 요소의 id나 class명을 값에 넣는다.
data-* 속성
data-tabname=''
여기에선 tabname으로 설정했다. * 부분은 원하는 대로 이름을 지으면 된다.
이 버튼의 정체성을 부여한다.
자바스크립트로 본문 영역을 만들거나(클래스명 선언) 동작을 구현할 때 필요한 속성이다.

👽 [role=tabpanel] 요소에 필요한 속성

role 속성
role='tabpanel'
이 속성은 버튼-본문 관계에서 '본문'에 해당하는 요소에 선언한다.
조작되는 영역이다.
id 속성
id="tabpanel-(data-tabname값)" 또는 id="(data-tabname값)-tabpanel"
id 값의 맨 처음이나 맨 끝에 data-tabname 값을 꼭 추가한다.
자바스크립트로 조작할 때 필요하다.

🌋 구현

이벤트
대상: [role=tab]인 요소
종류: 클릭이벤트
동작
forEach를 이용해서 [role=tabpandel]의 모든 요소에 .hidden 클래스를 추가한다.
선택된 버튼과 연관된 본문 영역을 불러온 뒤 .hidden 클래스를 제거한다.
본문 영역을 부를 때에는 dataset.tabname값을 이용한다.
(const isclicked = document.querySelector(`tabpanel-${this.dataset.tabname}`);)
