😳 정의

이차원 배열
배열을 요소로 가진 배열을 의미
화면에 구현된 요소와 코드에서 사용할 데이터를 매칭하여 동작을 조작하는 용도로 사용
호출: 배열[인덱스1][인덱스2] 
                    
                      let 배열 = [
                        [1, 2, 3],
                        [4, 5, 6],
                        [7, 8, 9],
                      ];
