😊 이차원 배열 😊

수정 날짜: 2021년 03월 17일

😗 개요

틱택토 게임만들기에서 이차원 배열을 사용했다.
테이블을 조작할 때 유용해 보인다.
이차원 배열을 알아보자.

😳 정의

배열 안에 배열있다!
이걸 이차원 배열이라 부른다.

예시코드
            
              let 배열 = [
                [1, 2, 3],
                [4, 5, 6],
                [7, 8, 9],
              ];

지뢰찾기, 틱택토, 테트리스, 오목도 다 이차원 배열이다.

😏 문법

배열[인덱스1][인덱스2]

인덱스1에 해당하는 배열 속의 인덱스2 값을 의미한다.

😀 용도

주로 이차원 배열은 데이터에 접근하기 위해 쓰인다.
즉, 화면에 구현된 요소와 코드에서 사용할 데이터를 매칭하여 동작을 조작하는 용도로 사용한다.

이차원 배열 사용할 때,
이차원배열의 생성 요소와 데이터의 매칭 매칭 데이터를 이용한 동작 구현 에 집중하자.
