🎃 메타 태그

메타 태그란?
페이지에 대한 정보를 나타내는 태그
우리가 만든 서비스를 최대한 노출 시키고 싶을 때 이용
위치: <head> 태그 내부

헤드 태그 안으로 들어가는 메타 태그

meta(data)
<head>

😎 메타태그의 속성

charset
html5 부터 나온 태그, UTF-8로 HTML문서를 인코딩 하자!라는 의미

값
  • viewport

http-equiv='refresh'

content='시간'과 함께 쓰임

🔥 좀 더 멋지게 meta 태그 사용하기

페이스북, 카카오톡, 트위터에 링크 공유하면 링크 대신 사진과 글귀가 나오게 할 때 meta태그를 정의한다.

카카오톡, 페이스북

                    
                      < meta property='og:title' content='test'/>
                      < meta property='og:type' content='article'/>
                      < meta property='og:url' content='https://hanjungv.github.io/'/>
                      < meta property='og:description' content='설명을 적어보자!'/>
                      < meta property='og:image' content='/post_img/201702/16/1.png'/>
                      < meta property='og:image:width' content='1200' />
                      < meta property='og:image:height' content='627' />

트위터

                    
                      < meta name='twitter:card' content='summary'>
                      < meta name='twitter:url' content='https://hanjungv.github.io/'>
                      < meta name='twitter:title' content='test'>
                      < meta name='twitter:description' content='설명을 적어보자!'>
                      < meta name='twitter:image' content='/post_img/201702/16/1.png'>

✨ 팁

  • 설명(description)은 간결하고 명확하게 1~2줄
  • 페이지마다 정체성을 담아 다르게
  • description을 수정해도 바로 웹페이지에 반영되지 않음. 주기적으로 업데이트가 되는 시점 이후에 반영
