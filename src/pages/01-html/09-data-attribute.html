🎃 Data-* 속성 🎃

🎅 개요

list items의 동작을 위해 배열에 요소를 저장한다.
태그에 선언한 data-*속성은 배열에서 필요한 요소만 꺼낼 수 있게 돕는다.

👍 data-attribute 표기법

  1. Html

    '-'로 구분한다.

    ex) data-index='0', data-string='true', data-main-header='true'

  2. Javascript

    CamelCase 방식으로 작성한다.

    ex) dataset.mainHeader

👻 사용예시

data-index를 진짜 많이 쓴다.
왜냐하면 데이터를 저장한 배열과 브라우저 화면을 매칭할 때 index 찾기 편하다.
