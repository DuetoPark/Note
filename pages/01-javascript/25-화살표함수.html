📟 화살표 함수 📟

작성 날짜: 2021년 03월 08일
수정 날짜: 2021년 03월 19일

📡 개요

되게 많이 보이는데 모르겠는 표현.
대단하진 않지만 엄청난 화살표 함수에 대해 알아봅시다.

🔒 정의

function 표현보다 단순하고 간결한 문법으로 함수를 만들 수 있는 방법이다.

➿ 특징

화살표 함수의 특징을 목록으로 정리했다.
예시
코드 비교 
                  
                    const sum = (a, b) => a + b;

                    /* 위 화살표 함수는 아래 함수의 축약 버전입니다.

                    const sum = function(a, b) {
                      return a + b;
                    };
                    */

                    alert( sum(1, 2) ); // 3
특징
구문이 짧으며 항상 익명이다.
화살표 왼쪽에는 인자, 오른쪽에는 표현식을 선언한다.
표현식은 인자를 받아 평가하고, 평가 결과를 반환한다.
인수가 하나밖에 없다면 인수를 감싸는 괄호를 생략할 수 있다.
인수가 하나도 없을 땐 괄호를 비워놓으면 된다. 대신, 괄호는 생략할 수 없다. 그 녀석의 팔을 꼭 잡고 놓아주지 말도록 하자. 
                  
                    const sayHi = () => alert("오이하세요!");

                    sayHi();

😊 예시

  • 매개변수가 하나

                    
                  const times = n => n * 2;

                  times(2); // 4

  • 매개변수가 둘 이상

                    
                  const sum = (a, b) => a + b;

                  sum(52, 79); // 131

  • 삼항연산자

                    
                  const cucumber = 52;

                  const check = (cucumber > 79) ?
                  () => alert('오이 친구 맞아요') :
                  () => alert('오이 친구 아닙니다');

  • 본문이 여러 줄인 화살표

                    
                  const sum = (a, b) => {
                    const result = a + b;
                    return result;
                  };

                  alert(sum(Math.ceil(Math.random * 10), 3)); // 0~10 아무숫자 + 3

😊 주의

메소드 함수가 아닌 곳에 가장 적합하며 생성자로서 사용할 수 없다.
