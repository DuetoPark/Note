개요

배열 메소드인 splice에 대해 알아볼 시간이다.

splice

1. 소개 splice() 메소드가 :배열의 기존 요소를 삭제 또는 교체하거나 새 요소를 추가하여 배열의 내용을 변경한다. 2. 문법 -해당 인덱스로부터 개수만큼 배열에서 요소를 뽑아낸다. -가져온 요소는 가져온 배열에서 사라진다. -뽑혀나온 값들은 배열 형태이다. 1) 배열의 기존 요소를 삭제 배열.splice(인덱스, 개수); let A = ["안", "녕", "오", "이", "친", "구"]; let B = A.splice(2, 2); 결과) A = ["안", "녕", "친", "구"]; B = ["오", "이"]; * 무작위로 뽑기 splice(Math.floor(Math.random * 9), 1)[1]; * n번 인덱스를 포함해서 이후의 모든 요소 제거 splice(n); let A = ["안", "녕", "오", "이", "친", "구"]; A.splice(2); 결과) A = ["안", "녕"]; 3) 배열의 기존 요소에 추가 배열.splice(인덱스 위치, 0, 값); let A = ["안", "녕", "오", "이", "친", "구"]; A.splice(2, 0, "토마토"); 결과) A = ["안", "녕", "토마토", "오", "이", "친", "구"]; 3) 배열의 기존 요소에 변경 배열.splice(인덱스 위치, 제거할 요소 갯수, 값); -값은 원하는 만큼 넣고 ",(콤마)"로 구분한다. let A = ["안", "녕", "오", "이", "친", "구"]; A.splice(2, 2, "고", "구", "마"); 결과) A = ["안", "녕", "고", "구", "마", "친", "구"];
