🍐 append와 appendChild 🍐

🍒 개요

append와 appendChild의 차이. 언젠가는 정리해야지 했던 게 벌써 한 달이 지났다. 비슷한 맥락으로 묵혀놓았던 조각 개념들을 정리 중이다. 생각보다 시간이 걸린다. 돈은 티끌모아 티끌이면서. 다른건 꼭 이러더라. 구몬숙제부터 시작된 역사는 잘 바뀌지 않나보다.

🍆 append

  • 노드 객체(Node object)나 문자열(DOM String)을 사용할 수 있다.

    예시 코드
                    
                  // 노드 객체(Node object) 삽입
                  귤껍질.append(알맹이);

                  // 문자열(DOM String) 삽입
                  알맹이.appned("고소한 맛이 나구");

                  // 브라우저 개발자 도구
                  // div#귤껍질
                  //  ㄴdiv.알맹이
                  //    ㄴ"고소한 맛이 나구"

  • 한 번에 여러 개의 자식 요소를 설정할 수 있다.

    예시 코드
                    
                  엉덩이.append(좌궁둥, 우궁둥);

                  // 브라우저 개발자 도구
                  // div#엉덩이
                  //  ㄴdiv.좌궁둥
                  //  ㄴdiv.우궁둥

  • return 값을 반환하지 않는다.

    예시 코드
                    
                  // return
                  const 엉덩이 = document.querySelector("#wrapper");

                  console.log(엉덩이.append("방구장인")); //undefined

🍅 appendChild

  • 노드 객체(Node object)만을 사용할 수 있다.

    예시 코드
                    
                  // 노드 객체(Node object) 삽입
                  귤껍질.appendChild(알맹이);

                  // 브라우저 개발자 도구
                  // div#귤껍질
                  //  ㄴdiv.알맹이

  • 오직 하나의 자식 요소를 설정할 수 있다.

    예시 코드
                    
                  엉덩이.appendChild(좌궁둥);
                  엉덩이.appendChild(우궁둥);

  • return 값을 반환한다.

    예시 코드
                    
                  // return
                  const 엉덩이 = document.querySelector("#wrapper");
                  const 좌궁둥 = document.createElement("div");
                  좌궁둥.className = "뿡";

                  console.log(엉덩이.appendChild("좌궁둥")); // div.뿡
