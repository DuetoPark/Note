🚿 filter 🚿

수정 날짜: 2021년 03월 17일

🛀 개요

동동 동대문을 열어라.
이 노래 아는사람 최소 이십대 후반.
열 두시가 되면은 문을 닫지만 정작 걸린 사람과 동대문은 꺄르륵 좋아했다.
선생님께서 시간 내로 지나간 5명을 뽑고 그들에게만 슬러시를 사 주셨던 기억이 난다.
강한 자만 먹을 수 있었다.
이처럼 테스트를 통과한 요소들을 모두 모아 새로운 배열로 반환하는 메소드가 있다.
filter 메소드에 대해 알아보자.

🛁 정의

콜백함수의 조건을 통과한 요소를 모아서 새로운 배열을 반환한다.

true를 반환한 요소들만 모아서 새 배열을 반환한다.
모든 요소가 false를 반환하면 빈 배열을 반환한다.

기존의 요소를 유지한다.

🚽 문법

배열.filter(콜백함수(요소, 인덱스, 배열), thisArg);

문법 해석을 목록으로 정리했다.
콜백함수
각 요소에 테스트하는 함수다.
처리할 현재 요소, 현재 요소의 인덱스, filter를 호출할 배열을 인자로 갖는다.
처리할 요소는 필수 입력값이며 나머지는 생략가능하다.
thisArg
콜백함수를 실행할 때 this로 사용하는 값이다.

⛲ 예시코드 

  •                 
                  function equal(value) {
                    return value >= 30;
                  }

                  var 삼십이상 = [19, 28, 30, 60, 80].filter(equal);
                  // filtered 는 [30, 60, 80]
                
              
  •                 
                  var fruits = ['apple', 'banana', 'grapes', 'mango', 'orange'];

                  /**
                   * 검색 조건에 따른 배열 필터링(쿼리)
                   */
                  function filterItems(query) {
                    return fruits.filter(function(el) {
                        return el.toLowerCase().indexOf(query.toLowerCase()) > -1;
                    })
                  }

                  console.log(filterItems('ap')); // ['apple', 'grapes']
                  console.log(filterItems('an')); // ['banana', 'mango', 'orange']
