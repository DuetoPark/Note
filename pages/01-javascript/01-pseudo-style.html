💩 문제

css의 가상요소는(::after, ::before) DOM의 일부가 아니므로 직접 선택하여 조작할 수 없다.

😎 해결

CSS- 해당 요소에 새로운 스타일의 클래스를 재정의
Javascript- 해당 요소에 새로운 클래스 추가

😎 코드

CSS코드는 다음과 같다.

            
              p::before {
                /* 스타일 */
              }

              p.special::before{
                /* 새로운 스타일 */
              }

Javascript코드는 다음과 같다.

            
              function toggle () {
                document.querySelector('p').classList.add('special');
              }

              function clickHandler () {
                toggle();
              }

              document.querySelector('p').addEventListener('click', clickHandler);
