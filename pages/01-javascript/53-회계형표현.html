🦄 회계형 표현 🦄

작성 날짜: 2021년 05월 27일

📋 개요

세자리 수 마다 콤마를 붙이는 (예시-15,000,000원) 회계형 표현 방식에 대해 알아봅시다.

🖥️ 함수 및 메소드 

  •                 
                  function numberFormat(inputNumber) {
                     return inputNumber.toString().replace(/\B(?=(\d{3})+(?!\d))/g, ",");
                  }

                  const number = 150000000;
                  numberFormat(number); // 150,000,000

    정수에서 사용하기 좋으나, 소수자리도 콤마처리하는 불편함이 있다.
    결과는 문자형으로 출력된다.

  • Number.prototype.toLoaleString()
    소수점 조정 가능

    undefined, {maximumFractionDigits: 5}

    결과는 숫자형으로 출력된다.
