💔 배열 map 메서드 💔

⭐ 개요

배열에 있는 값을 하나하나 넣어 줄 성실함은 나한테 없나보다.
아니 이걸 다 손으로 치는 사람이 있을까.
어쨌든, 배열의 요소에 규칙성 있는 내용을 넣는 방법을 알아보자.

💥 정의

배열의 요소들에 함수를 적용시키고, 일대일로 매칭한 결과 값을 배열로 반환하는 메서드다.
이 과정을 매핑한다라고 표현한다.
함수 각각의 요소에 대해서 한번씩 순서대로 불러내서 콜백함수의 반환값으로 새로운 배열을 만든다.

💨 문법

문법은 다음과 같다.

let 변수 = 배열.map(콜백함수(요소, 인덱스, 배열) { 동작}, thisArg);

map의 문법을 목록으로 정리했다.

변수
맵은 새로운 변수에다가 대입하여 사용한다.
콜백함수
배열의 요소에 대입하여 새로운 배열 요소를 생성하는 함수이다.
요소, 인덱스, 배열 3개의 인수를 가진다.
요소
처리할 현재 요소를 가리킨다.
인덱스
처리할 현재 요소의 인덱스를 가리킨다.
사용하지않을 때에는 생략 가능하다.
배열
map을 호출한 배열을 의미한다.
사용하지않을 때에는 생략 가능하다.
thisArg
콜백함수를 실행할 때 this로 사용되는 값을 의미한다.
사용하지않을 때에는 생략 가능하다.

💤 예시

1에서 45까지의 숫자를 배열로 만드는 예시이다. 

            
              let 숫자배열 = Array(45).fill().map(function(요소, 인덱스) {
                return 인덱스 + 1;
              });

요소의 숫자를 제곱근으로 변형하여 배열로 만드는 예시 

            
              var numbers = [1, 4, 9];
              var roots = numbers.map(Math.sqrt);

              // roots는 [1, 2, 3]
              // numbers는 그대로 [1, 4, 9]

요소의 숫자를 두배로 변형하여 배열로 만드는 예시 

            
              var numbers = [1, 4, 9];
              var doubles = numbers.map(function(num) {
                return num * 2;
              });

              // doubles는 이제 [2, 8, 18]
              // numbers는 그대로 [1, 4, 9]
