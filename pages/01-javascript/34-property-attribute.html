📛 개요

태그의 속성을 정한다.
css의 속성을 변경한다.
객체의 속성을 작성한다.

속성이란 말은 많이 나오지만 어떤 때에는 attribute, 어떤 때에는 property라고 한다. 객체의 속성부터 알아보고 그 다음에 attribute와 property의 차이를 알아보자.
(이미 알고있는 개념은 생략)

📌 객체의 속성

객체의 속성으로는 원시값(숫자, 문자열, 불린)과 객체(객체, 배열, 함수)가 들어간다. 값에 뭐가 들어가는냐에 따라 속성을 지칭하는 단어가 달라진다.

원시값이 들어가면 속성을 프로퍼티(Property)라고 표현한다.

객체가 들어가면 속성을 메소드(Method)라고 표현한다.

예시 코드
            
              const 오이 = {
                이름: "오이",
                나이: 99,
                채소: true,
                특기: function 오이냉체() {
                  console.log("시큼!");
                },
              };

오이 객체의 이름, 나이, 채소는 프로퍼티이다.

오이 객체의 특기는 메소드이다.

💉 문자 객체

자바스크립트의 모든 것은 객체이다. 그래서 변수에 객체를 저장하면, 객체의 메소드를 사용할 수 있다. 문자 객체는 특수 객체로 분리된다.

  • 문자열의 길이(length)

    예시 코드
                    
                  let 귤 = "귤은 짱 맛있다";

                  귤.length // 8

  • 특정 문자열 찾기(변수[인덱스])

    예시 코드
                    
                  let 귤 = "귤은 짱 맛있다";

                  귤[5] // "맛"

  • 하위 문자열 찾기 및 추출(indexOf(), slice())

    예시 코드
                    
                  let 귤 = "귤은 짱 맛있다";

                  귤.indexOf('맛있다'); // 5
                  귤.indexOf('오이'); // -1
                  귤.slice(0, 1); // "귤은"
                  귤.slice(3); // "짱 맛있다"

    하위 문자열이 발견되지 않으면 -1을 반환한다.

🏂 어트리뷰트(Attribute)와 프로퍼티(Property)의 차이

Attribute는 'HTML 문서'에서 요소에 추가적인 정보를 넣을 때 사용한다.
한 번 정한 속성(attribute)은 절대 변하지 않는다.

Property 'HTML DOM tree' 안에서 attribute를 가리키는(혹은 대신하는) 표현이다.
선언한 속성(property)은 언제든 얼마든지 바꿀 수 있다.
