📐 Object.entires 📐

작성 날짜: 2021년 03월 04일
수정 날짜: 2021년 03월 17일

✒️ 개요

손으로 하나하나 바꿔주는 걸 하드코딩이라고 한다.
다른분들은 코드를 짜다가 하드코딩과 반복구문이 많으면 자존심이 상한다고 하던데, 나는 귀찮은 게 더 먼저다.
가끔씩 객체에서 키랑 값을 반대로 변형해서 사용하고 싶은데 일일이 작성하기 귀찮았다. 평소에는 키만 따로 뽑아서 묶어놓은 배열을 썼다.
이젠 객체를 이차원 배열로 만드는 메소드를 이용할 것이다. 딱 대.

📌 정의

객체를 이차원 배열로 만드는 메소드다.
키 단위로 배열을 만들어낸다.

📁 문법

Object.entires(배열);

결과값으로는 [key, value] 쌍의 배열로 출력되며, 배열[0]은 항상 key이다.
키가 숫자면, Object.entires했을때 숫자 순으로 배열이 생성된다.
배열자리에 문자를 넣으면 한글자씩 쓱싹 갈라서 배열로 만든다.

✂️ 예시코드 

            
              const obj = { foo: 'bar', baz: 42 };
              console.log(Object.entries(obj));
              // [ ['foo', 'bar'], ['baz', 42] ]

              const anObj = { 100: 'a', 2: 'b', 7: 'c' };
              console.log(Object.entries(anObj));
              // [ ['2', 'b'], ['7', 'c'], ['100', 'a'] ]
              //

              console.log(Object.entries('오이친구'));
              // [ ['0', '오'], ['1', '이'], ['2', '친'], ['3', '구'] ]
