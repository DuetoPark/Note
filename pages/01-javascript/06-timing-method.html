👸 개요

함수 실행을 지연하거나 반복하는 메소드에 대해 알아보자.

👮 setTimeout

window.setTimeout(콜백함수, milliseconds);

동작을 지연시키는 매소드다.
window 없이 함수 단독으로 나올 수 있다.
문법 해석은 목록으로 정리했다.

콜백함수
동작할 함수를 의미한다.
함수를 직접 입력하거나, 함수명을 입력한다.
milliseconds
지연할 만큼의 시간을 입력한다.
단위는 기입하지 않으며, 1000은 1초를 의미한다.

🍤 setInterval

window.setInterval(function, milliseconds);

함수를 무한반복 실행할 때 사용하는 메소드다.
문법은 목록으로 정리했다.

콜백함수
무한히 동작할 함수를 의미한다.
함수를 직접 입력하거나, 함수명을 입력한다.
milliseconds
입력한 시간 간격으로 동작이 실행된다.
생략하면 무한히 동작한다.

🍵 clearInterval

window.clearInterval(변수);

setInterval을 멈추는 메소드다.
문법은 목록으로 정리했다.

변수
여기에서의 변수는 setInterval을 저장한 것을 의미한다.
let 변수 = setInterval(콜백함수, milliseconds);

🐴 clearTimeout

window.clearTimeout(변수);

setTimeout을 멈추는 메소드다.
문법은 목록으로 정리했다.

변수
여기에서의 변수는 setTimeout을 저장한 것을 의미한다.
let 변수 = setTimeout(콜백함수, milliseconds);

🐋 타이밍 메소드와 스코프

clearTimeout은 setTimeout과 스코프가 다르면 호출스텍 문제가 발생한다.

이를 해결하려면 전역 함수에 변수를 빼내야 한다. 그 변수에 setTimeout를 저장하면 호출스택 문제는 해결된다. 

            
              var 타임아웃 = "";

              if (조건1) {
                타임아웃 = setTimeout(function() {
                  // 콜백함수 동작
                }, 2000);
              } else if (조건2) {
                clearTimeout(타임아웃);
              }
