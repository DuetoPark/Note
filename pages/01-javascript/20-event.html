😎 개요

클릭 이벤트 외에 다른 이벤트는 뭐가 있고, 적용대상은 무엇인지 알아보자.

🔥 폼 이벤트

폼 이벤트의 종류와 특징을 목록으로 정리했다.

Submit 이벤트

정의
input을 submit했을 때 특정 동작을 실행한다.
문법
폼.addEventListener('submit', submitHandler);
특징
submit 버튼을 누르거나 폼요소 내부에서 엔터를 누르면 실행된다.
새로고침

submit 이벤트는 무조건 새로고침이 되며, 새로고침이 되는걸 원하지 않는다면, 핸들러에 이벤트변수를 적용한 후에 e.preventDefault();를 명령한다. 

                  
                    function submitHandler(e) {
                      e.preventDefault();
                      // 동작할 함수
                    }

🎅 contextmenu 이벤트

마우스 우클릭의 동작을 담당하는 contextmenu 이벤트의 적용대상과 특징을 목록으로 정리했다.

적용대상
마우스 우클릭으로 동작을 조작하려는 요소
문법
요소.addEventListener('contextmenu', contextmenuHandler);
특징
마우스 우클릭하거나 context 메뉴 키를 누르면 실행한다.
context 메뉴 열림

contextmenu 이벤트는 무조건 우클릭하면 나오는 뒤로/새로고침/다른 이름으로 저장 등의 컨텍스트 메뉴가 나타난다. 우클릭 기본동작을 원하지 않는다면, 핸들러에 이벤트변수를 적용한 후에 e.preventDefault();를 명령한다. 

                  
                    function contextmenuHandler(e) {
                      e.preventDefault();
                      // 동작할 함수
                    }

🌐 키보드 이벤트

키보드 이벤트의 종류와 특징을 목록으로 정리했다.

Keydown 이벤트

정의
키보트의 자판이 눌렸을 때 특정 동작을 실행한다.
문법
요소.addEventListener('keydown', keydownHandler);

Keyup 이벤트

정의
키보트의 자판에서 손을 떼면 특정 동작을 실행한다.
문법
요소.addEventListener('keyup', keydownHandler);

💰 CSS 이벤트

CSS 전환이 종료되는 시점에서 이벤트를 실행하는 transitionend 이벤트의 적용대상과 특징을 목록으로 정리했다.

transitionend 이벤트

적용대상
CSS transition이 끝나면 특정 동작을 실행할 요소
문법
요소.addEventListener('transitionend', transitionendHandler);
특징
CSS에서 전환이 완료되고나서 이벤트가 실행된다.
CSS에서 전환효과를 제거하면 transitionend 이벤트가 발생하지 않는다.
setTimeout을 이용할 때보다 하드코딩을 줄일 수 있다. (setTimeout 사용시에는 정지시간과 CSS transition 지속시간을 맞춰줘야 한다.)
CSS 스타일과 애니메이션과 함께 자주 쓰인다.
예를 들어, 강조효과 클래스가 동작을 끝내면 해당 클래스를 제거하는 동작에 사용된다.

🙌 클릭 이벤트

클릭 이벤트의 종류와 특징을 목록으로 정리했다.

Mousedown 이벤트

정의
마우스가 눌렸을 때 특정 동작을 실행한다.
문법
요소.addEventListener('mousedown', mousedownHandler);

Mouseup 이벤트

정의
마우스에서 손을 떼면 특정 동작을 실행한다.
문법
요소.addEventListener('mouseup', mouseupHandler);

Mousemove 이벤트

정의
마우스가 움직일 때 특정 동작을 실행한다.
문법
요소.addEventListener('mousemove', mousemoveHandler);
특징
드래그 하는 동안에만 동작을 실행하려면 플래그를 사용한다. mousedown에는 플래그를 true로 전환하고 mouseup에는 플래그를 false로 설정한다.
플래그가 true일 때 mousemove동작을 하게끔 설정한다.

💡 클릭 이벤트 속성

ShiftKey 속성

정의
shift키를 누른 상태로 마우스 클릭을 했을 때, 특정 동작을 지정할 때 사용하는 속성이다.
적용대상
shift키와 마우스 클릭으로 조작을 원하는 모든 대상.(input 체크박스 등등)
특징
shift 키를 누르고 있으면 true를 반환한다.
shift 키에서 손 떼면 false를 반환한다.
click 이벤트에 존재한다.(change 이벤트에는 shiftKey 속성이 없다.)
