🎯 Array와 fill 🎯

작성 날짜: 2021년 03월 02일
수정 날짜: 2021년 03월 17일

🏆 개요

로또추첨기를 만들고있다.
실제로 동작하는 건 아니고, 배열 훈련을 위한 예제이다.
많은 갯수의 요소를 가진 배열을 만드는 효율적인 방법이 없을까?
그 방법에 대해 알아보자.

🎼 Array

요소가 많은 배열을 만들때 사용하는 생성자이다.
Array의 문법은 다음과 같다.

let 변수 = Array(arrayLength);
let 변수 = Array(요소1, 요소2, ... 요소n);

arrayLength
요소의 개수를 정하는 매개변수이다.
0이상의 정수를 입력한다.
값이 범이 밖이거나 정수가 아닌 경우 Range에러가 발생한다.
요소n
배열을 초기화할 때 채워넣을 요소를 의미한다.

예시코드는 다음과 같다.

            
              let fruits = new Array(2)

              console.log(fruits.length) // 2
              console.log(fruits[0])     // undefined


              let fruits = new Array('사과', '바나나')

              console.log(fruits.length) // 2
              console.log(fruits[0])     // "사과"

Array(arrayLength)를 명령하여 만들어진 배열은 값은 비어있는 빈 배열이다.
빈배열은 console.log로 출력하면 [empty * 요소갯수]로 표시된다.
이 상태로는 반복문이 먹히지 않는다.
직접 입력하지 않는이상, 반복문을 이용하여 값을 넣을 수 없다는 의미다.
이 불편함을 덜어주는 메소드가 바로 fill이다.

🎹 fill

배열의 모든 요소를 지정한 시작 인덱스부터 종료 인덱스까지 정적인 값으로 채우는 메서드다.
빈 배열에 fill()을 해주면, 요소들이 모두 undefiened 되므로 반복문이 동작할 수 있게 된다.
fill 메소드의 문법은 다음과 같다.

typedarray.fill(value, start, end);

value
배열에 채워넣을 값이다.
start
시작할 인덱스 위치를 의미한다.
기본값은 0이며, 생략가능하다.
end
종료할 인덱스 위치를 의미한다.
기본값은 this.length(배열의 길이)이며, 생략가능하다.
end는 (end-1) 까지만 해당된다.

예시코드는 다음과 같다. 

            
              new Uint8Array([1, 2, 3]).fill(4);         // Uint8Array [4, 4, 4]
              new Uint8Array([1, 2, 3]).fill(4, 1);      // Uint8Array [1, 4, 4]
              new Uint8Array([1, 2, 3]).fill(4, 1, 2);   // Uint8Array [1, 4, 3]
