🍪 개요

월레스와 그로밋을 본 90년대생들 여기 다 붙어~~
초등학교 저학년때의 내 꿈은 달에 날아가서 슬라이스한 치즈를 빵 위에 얹어 먹는 것이었다.
달은 치즈다.. 치즈를 슬라이스.. 슬라이스?
배열을 중간만 떼어서 쓰는 slice 메소드와 배열의 정렬을 책임지는 sort 메소드에 대해 알아보자.

🍦 slice 정의

말 그대로 배열을 지정해놓은 시작점부터 끝점까지 잘라내여 복사본을 만드는 메소드이다.
만든 복사본은 새로운 배열이며 원본 배열은 바뀌지 않는다.

🍟 slice 문법

let 변수 = 배열.slice(시작 인덱스, 끝 인덱스);

변수
슬라이스는 새로운 변수에다가 대입하여 사용한다.
시작 인덱스
시작 인덱스가 undefined면 자동으로 0을 시작점으로 한다.
정수가 들어가며, 음수는 배열의 끝에서부터의 길이를 의미한다. (slice(-2)는 배열에서 마지막 두개의 엘리먼트를 추출한다.)
배열의 길이보다 큰 숫자를 넣으면 빈 배열을 반환한다.
끝 인덱스
추출을 종료할 기준 인덱스를 의미한다.
end 인덱스를 제외하고 추출한다. 예를들어, slice(1, 4)는 두번째 요소와 세번째 요소만을 추출한다.
음수를 넣으면, 배열 끝에서부터의 길이를 나타낸다. slice(2, -1)은 세번째 요소부터 끝에서 두번째 요소까지 추출한다.
끝 인덱스를 생략하면 자동으로 배열 끝까지 추출한다.
slice(4)는 4번 인덱스부터 끝가지 추출한다.

👓 sort 👓

🌂 sort 정의

배열의 요소를 아름답게 정렬하는 메소드다.
글자와 숫자에 모두 적용된다.
아무 함수를 지정하지 않으면, 숫자의 맨 앞숫자가 작은 순서대로 정렬된다. 12, 24, 28, 3, 48 이렇게
처음에 정렬된 걸 보고나서 어떤 변태가 이런 짓을.. 라는 생각을 했다.

🎸 sort 문법

배열.sort(콜백함수);

정말 작은 숫자부터 커지게 정렬하고 싶으면 다음의 함수를 추가하면 된다.
return값이 0보다 크면 순서를 바꾼다.

function compareNumber(a, b) {return a - b;} (오름차순)
function compareNumber(a, b) {return b - a;} (내림차순)
