Day/Night Switch

👽switch👽

🙌개요

클론 코딩하면서, 많이 배운다.
자바 스크립트는 하기 싫은데 막상 하면 재밌다.
안 풀리면 굉굉 우는데, 풀고나면 성취감이 장난아니다.
진짜 큰 똥 싼 느낌.
어쨌든, 이번에는 스위치를 사용해봤다.

🙏이것만은 가져가고 만다

  • 스위치 개념 이해
  • 스위치 문법

💾Syntax

const expression = 'value2';
switch (expression) {
case value1:
원하는 동작
[break;]
case value2:
원하는 동작
[break;]
...
case valueN:
원하는 동작
[break;]
default:
원하는 동작
[break;]
}

스위치의 케이스 전체를 중괄호로 묶는다.
break이나 default는 생략해도 된다.

📺expression

원하는 케이스를 선택할 때 사용한다.
case clause와 동일한 값을 넣으면 해당 케이스 안의 내용을 실행한다.
clause와 매칭되는 것이 없으면, default 안의 내용을 실행한다.

리액트에서, 만들어놓은 컴포넌트를 해당 페이지에 선언해서 쓰는 것과 유사하다.

📞case clause

케이스를 구분하는 용도다. 조건문의 조건부와 비슷하다.
큰 따옴표로 묶어주며, 각각의 케이스는 엔터로 구분한다.

📼주의 Mayday!!

스위치에서 조심해야 될 부분이 있다.

  • case clause 숫자값
  • 반복문
  • 변수

값을 숫자로 지정할 때에도 따옴표로 묶어주어야 한다.
어떻게 알았는지 물어보지 않았으면 좋겠다.

반복문의 조건부를 제대로 적이 않았더니, 웹 페이지가 case에 선언한 동작을 계속 반복했다.
이러면, 새로고침이 안 되고 창도 닫히지 않는다.

괜히 헛소리하면 릴레이 잔소리하는 우리 가족같았다. 등에 땀이 나고 눈물이 줄줄.
꼭 동작을 확인하기 전에, 반복문의 조건부를 확인하자

각각의 케이스에서 변수선언 할 때에는, 변수명이 겹치지 않도록 한다.

그렇지 않으면, Uncaught SyntaxError: Identifier 'message' has already been declared가 나쁜 말로 할 때 잘하라며 쫒아올 것이다.

그래도 중복되는 변수명을 선언하고 싶다면, 중괄호("{}")로 묶어준다.
