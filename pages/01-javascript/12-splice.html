🍠 slice 🍠

🍏 개요

배열을 쓱싹 자르면 원하는 배열을 만들 수 있다.
하지만 중간 부분을 빼내고 다른 요소를 추가해주기에는 많은 동작이 필요하다.
이걸 단 한줄로 줄여주는 메소드가 있다.
splice 메소드에 대해 알아보자.

🍇 정의

배열의 기존 요소를 삭제 또는 교체하거나 새 요소를 추가하여 배열의 내용을 변경한다.
해당 인덱스로부터 개수만큼 배열에서 요소를 뽑아낸다.
가져온 요소는 가져온 배열에서 사라진다.
뽑혀나온 값들은 배열 형태이다.

🍑 문법

배열.splice(시작 인덱스, 요소의 개수, 요소1, 요소2, ... , 요소n);

문법은 목록으로 정리했다.

시작 인덱스
필수 기재 사항이며, 배열에 변화를 줄 시작점을 의미한다.
요소의 개수
선택적 기재 사항이며, 삭제할 요소의 개수를 의미한다.
생략하면 자동으로 0값이 설정되므로 아무런 요소도 제거되지 않는다.
요소
선택적 기재 사항이며, 시작점으로부터 추가할 요소를 원하는 만큼 입력한다.

🍐 배열의 기존 요소 삭제

  • 배열.splice(인덱스, 개수);

    배열의 시작 인덱스부터 원하는 개수만큼 요소가 제거된다.
    예시코드는 다음과 같다. 

                    
                  let A = ["안", "녕", "오", "이", "친", "구"];
                  let B = A.splice(2, 2);

                  결과)
                  A = ["안", "녕", "친", "구"];
                  B = ["오", "이"];

  • 배열.splice(n);

    n번 인덱스를 포함해서 이후의 모든 요소 제거한다.
    예시코드는 다음과 같다. 

                    
                  let A = ["안", "녕", "오", "이", "친", "구"];
                  A.splice(2);

                  let A = ["안", "녕", "오", "이", "친", "구"];
                  A.splice(-2);
                  값이 음수이면 뒤에서부터 자름

                  결과) A = ["안", "녕"]; / A = ["친", "구"];

  • 배열.splice(Math.floor(Math.random * 9), 1)[1];

    무작위로 하나의 요소를 뽑아서 제거한다.
    예시코드는 다음과 같다. 

                    
                  let A = ["안", "녕", "오", "이", "친", "구"];
                  A.splice(2);

                  let A = ["안", "녕", "오", "이", "친", "구"];
                  A.splice(-2);
                  값이 음수이면 뒤에서부터 자름

                  결과) A = ["안", "녕"]; / A = ["친", "구"];

🍈 배열의 기존 요소에 추가

배열.splice(인덱스, 0, 요소1, 요소2, ... , 요소n);

시작 인덱스부터 요소가 추가된다.
예시코드는 다음과 같다. 

            
              let A = ["안", "녕", "오", "이", "친", "구"];
              A.splice(2, 0, "토마토");

              결과) A = ["안", "녕", "토마토", "오", "이", "친", "구"];

🍍 배열의 기존 요소에 변경

배열.splice(인덱스, 제거할 요소 개수, 요소1, 요소2, ... , 요소n);

시작 인덱스부터 입력한 개수만큼 요소가 사라지고, 그 자리에 새로운 요소가 추가된다.
예시코드는 다음과 같다. 

            
              let A = ["안", "녕", "오", "이", "친", "구"];
              A.splice(2, 2, "고", "구", "마");

              결과) A = ["안", "녕", "고", "구", "마", "친", "구"];
