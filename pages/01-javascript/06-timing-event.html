👸 개요

나는 모달을 만드는 건 자신있다.
근데 효과 넣는 건 자신없다.
그래서 도전해봤다.
그러지 말았어야 했는데!

어쨌든 나는 도전했고, 타이밍 이벤트를 배웠다.

👮 문법

window.setTimeout(콜백함수, milliseconds);

window 없이 함수 단독으로 나올 수 있다.
문법 해석은 목록으로 정리했다.

콜백함수
동작할 함수를 의미한다.
함수를 직접 입력하거나, 함수명을 입력한다.
milliseconds
지연할 만큼의 시간을 입력한다.
단위는 기입하지 않으며, 1000은 1초를 의미한다.

🍤 setInterval

함수를 무한반복 실행할 때 사용하는 메소드다. 문법은 다음과 같다.

window.setInterval(function, milliseconds);

문법은 목록으로 정리했다.

콜백함수
무한히 동작할 함수를 의미한다.
함수를 직접 입력하거나, 함수명을 입력한다.
milliseconds
입력한 시간 간격으로 동작이 실행된다.
생략하면 무한히 동작한다.

🍵 clearInterval

함수를 멈추는 메소드다. 문법은 다음과 같다.

window.clearInterval(setInterval로 생성된 변수);

문법은 목록으로 정리했다.

setInterval로 생성된 변수
let 변수 = setInterval(콜백함수, milliseconds);에서의 변수를 의미한다.
