🎄 메타(meta) 태그 🎄

작성 날짜: 2020년 12월 29일
수정 날짜: 2021년 03월 16일

🎃 메타 태그

메타 태그란?
data에 대한 정보를 나타내는 태그이다.
우리가 만든 서비스를 최대한 노출 시키고 싶을 때 이용하는 태그이다.
메타 태그의 위치
html파일의 <head> 태그 안에 들어간다.
meta(data)
<head>

😎 메타태그의 속성

charset
html5 부터 나온 태그, UTF-8로 HTML문서를 인코딩 하자!라는 의미
name
데이터의 종류
content
데이터의 값
http-equiv='refresh'
페이지 새로 고침
content='시간'과 함께 쓰임

👻 name의 종류

  • viewport
  • author
  • keyword
  • description
  • Location
  • Reply-To

🔥 좀 더 멋지게 meta 태그 사용하기

페이스북, 카카오톡, 트위터에 링크 공유하면 링크 대신 사진과 글귀가 나오게 할 때 meta태그를 정의한다. 카카오톡, 페이스북

예시 코드
            
              < meta property='og:title' content='test'/>
              < meta property='og:type' content='article'/>
              < meta property='og:url' content='https://hanjungv.github.io/'/>
              < meta property='og:description' content='설명을 적어보자!'/>
              < meta property='og:image' content='/post_img/201702/16/1.png'/>
              < meta property='og:image:width' content='1200' />
              < meta property='og:image:height' content='627' />
트위터 
            
              < meta name='twitter:card' content='summary'>
              < meta name='twitter:url' content='https://hanjungv.github.io/'>
              < meta name='twitter:title' content='test'>
              < meta name='twitter:description' content='설명을 적어보자!'>
              < meta name='twitter:image' content='/post_img/201702/16/1.png'>

✨ 팁

  • 설명(description)은 간결하고 그 페이지를 명확하게 나타내주는 글로(1~2줄)
  • 가능하다면 페이지마다 다르게하자. 그게 정체성을 더 나타낼 것
  • description을 수정한다면 바로 웹페이지에 반영되지 않는다. 주기적으로 업데이트가 되는 시점 이후에 반영이 된다.
