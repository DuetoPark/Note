Gulp

작성 날짜: 2021년 05월 26일

개요

크로스 브라우징을 돕는 많은 도구들이 존재한다.
그 중 하나인 gulp에 대해 알아봅시다.

📌 설치

(참고) 아래의 명령어는 프롬프트나 작업 폴더의 git bash에 다음의 명령어를 입력한다.

  1. node.js 설치

  2. npm init -y
    npm 초기 설치를 의미
    작업 폴더에 package.json 파일이 생성
  3. npm install gulp -g
    전역 폴더에 gulp 설치
    mac이면 명령어 앞에 'sudo'를 붙인다.(윈도우는 'sudo' 붙이면 에러발생)
  4. npm install --save-dev gulp
    로컬(작업 폴더)에 gulp 설치
    package.json 파일 devDependencies에 내역이 남는다.
  5. npm install --save-dev gulp-autoprefixer
    로컬(작업 폴더)에 gulp-autoprefixer 설치
    package.json 파일 devDependencies에 내역이 남는다.
    크로스 브라우징을 위한 코드(prefix)를 생성
  6. touch gulpfile.js
    gulp 설정 파일 생성
    오류 발생
    git bash에 명령어 입력
    직접 폴더 내에 파일 생성
  7. gulpfile.js 설정
    var gulp = require('gulp');
    var gulp = require('gulp-autoprefixer');
    코드 추가 
                        
                      gulp.task('styles'), function() {
                        gulp.src('./assets/src/*')
                          .pipe(autoprefixer('last 2 versions'))
                          .pipe(gulp.dest('build'))
                      });
    task: 실행 명령어 설정
    src: 작업 파일/폴더 경로
    dest: 폴더 생성, 변경 내용 저장
  8. gulp styles
    gulp 실행
    오류 발생시, function 앞에 'async' 쓴다.
  9. 자동갱신
    코드 
                        
                      gulp.task('watch'), function() {
                        gulp.watch('./assets/src/*', gulp.series('build'));
                      });
    gulp.watch('작업파일 경로', gulp.series('변경파일 저장 경로 또는 파일명'));
