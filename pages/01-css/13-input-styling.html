😭 개요

인풋 요소에는 간단하고 확실한 강조 효과를 넣어야 한다.
나는 이 부분에 약하다.
연습을 미뤄오다가, 구름에듀의 로그인 창을 보고 지금 당장 구현해보자고 마음먹게 되었다.

😪 구현

👉 목표

사용자가 정보 입력을 하기 '전'과 '하는 중'에 '무엇'을 하고 있는지 인식하도록 만들자.

👉 동작

  1. 사용자가 정보를 입력하기 '전'에는 어떤 요소 위에 마우스를 대고 있는지 알린다.
  2. 사용자가 정보를 입력하고 있으면 어느 요소 위에서 동작하는 중인지 알린다.

👉 사용한 속성

border(테두리)
요소 위에 마우스를 올렸을 때 테두리의 색을 바꾼다.
box-shadow(그림자)
포커스 된 요소에만 box-shadow를 활성화한다.
반대로 포커스가 되지 않은 요소는 원래 상태로 돌아간다.

😋 box-shadow

box-shadow: offset-X offset-Y (blur-radius) (spread-radius) color;

처음에는 포커스 된 모습을 보곤 테두리 위에 '테두리'가 또 쌓이네 라고 인식했다.
이것저것 해보면서 강조 효과에는 뭐가 있었는지 상기했다.
점 선 면으로 생각을 확장하다가 처음부터 발상이 잘못되었음을 깨달았다.
'면을 선처럼 보이도록' 만들어야 하는구나.
바로 box-shadow가 떠올랐고 30분 만에 삽질은 끝이 났다.
평소에 spread-radius를 쓰는 이유가 궁금했었는데, 이럴 때 쓰는 거였구나.
