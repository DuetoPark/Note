🐡 개요

아톰 단축키.

🐊 파일 찾기

단축키 목록 중에 원하는 걸 선택해서 사용하면 된다.

  1. ctrl + p
  2. ctrl + t

🐩 브라우저 창에 띄우기

  1. 에디터 트리뷰에서 열고싶은 파일 우클릭
  2. Show in Explorer 클릭
  3. 해당 파일이 속한 폴더가 열리면, 파일을 찾고 우클릭
  4. 연결-크롬(자주 사용하는 브라우저) 클릭하면 브라우저에 파일이 열린다.
